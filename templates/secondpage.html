Django (อ่านว่าจังโก้ หรือแจงโก้ โดยไม่ออกเสียงตัว D)
เป็น framework ที่ใช้ในการสร้าง Web Application ในฝั่งของ Back End
ที่พัฒนาด้วยภาษา Python โดยในตัว framework จะมีส่วนประกอบทุกอย่างที่จำเป็น
ตั้งแต่การเชื่อมต่อฐานข้อมูล ไปจนถึงการ render ข้อมูลออกมาให้ฝั่ง Front End
แสดงผลข้อมูลเหล่านั้นได้ ซึ่ง framework ในรูปแบบนี้ในภาษาอื่นๆ เช่น Ruby on rails
สำหรับภาษา Ruby, Play Framework สำหรับภาษา Java หรือ Scala,
Groovy on Grails สำหรับภาษา Groovy, Laravel สำหรัาษา PHP,
หรือ Express สำหรับภาษา Javascript ของ Node.js เป็นต้น
ซึ่งข้อมูลที่อ้างอิงมาจาก www.hotframeworks.com จะเห็นว่า Django
มีการใช้งานอย่างแพร่หลายมาก
